# Ronald Nubah Portfolio Cleanup Blueprint

## Overall Direction

Position the site as a UI/UX and product design portfolio first. The current version says "apps, websites, logos, and more," which feels broad. A stronger direction is:

> UI/UX designer creating clear, useful digital experiences through research, prototyping, and thoughtful visual design.

## Homepage Structure

1. Hero
2. Featured Projects
3. About Snapshot
4. Skills / Tools
5. Contact

## Navigation

Use:

- Work
- About
- Contact

Remove "Services" from the main navigation for now. For a portfolio, "Work" is stronger than "Services" because the goal is to show proof.

## Hero Section

Current issue:
The background image is too bright and the hero text is low contrast, so the most important message is hard to read.

Recommended hero copy:

**Ronald Nubah**

UI/UX designer building thoughtful digital products, prototypes, and brand experiences.

Button text:

- View Work
- Contact Me

Design notes:

- Use a cleaner, darker overlay on the image or switch to a solid light background.
- Keep the hero text left aligned instead of centered.
- Make the headline easy to read immediately.
- Avoid all-caps paragraph text.

## Projects Section

Rename "PROJECTS" to:

**Featured Work**

Each project card should include:

- Project image
- Project title
- One-sentence summary
- Tags, such as UI Design, UX Research, Figma, Prototype
- Button or link: View Project

Suggested project card format:

**PODLY**

A music recommendation app concept focused on helping users discover audio based on mood, taste, and listening habits.

Tags: Figma, Mobile App, UX/UI, Prototype

## Visual Cleanup

- Use one clean font family for most text.
- Keep the bold display font only for your name or major section headings.
- Increase contrast between text and background.
- Use consistent spacing between sections.
- Make all project images the same ratio.
- Avoid using unrelated stock images in the project grid unless they directly represent the project.
- Make the project thumbnails feel like portfolio case study covers, not random images.

## Copy Cleanup

Replace:

"I create prototype apps, websites, logos, and more!"

With:

"I design digital experiences in Figma, from early concepts and user flows to polished interactive prototypes."

## Recommended About Preview

I’m Ronald Nubah, a UI/UX designer focused on creating clean, approachable digital products. My work combines visual design, prototyping, and problem solving to turn ideas into experiences people can understand and use.

## Contact Section

Use a simple ending:

**Let’s build something clear and useful.**

For portfolio inquiries, collaborations, or feedback, reach out anytime.

Buttons:

- Email Me
- View Resume

## Priority Fixes

1. Fix hero text contrast.
2. Change navigation to Work, About, Contact.
3. Replace "Services" framing with portfolio/work framing.
4. Make project cards consistent.
5. Rewrite project descriptions so each one explains what the project is and what you did.
6. Remove or replace generic stock imagery.
